## Run Locally


Clone the repository

```bash
  git clone https://github.com/anuragtomar2510/kbc-app.git
```

Go to the project directory

```bash
  cd kbc-app
```


Install dependencies

```bash
  npm install
```

start server 

```bash
  npm start
```
